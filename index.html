Modal-Form Demo

This is the demo page for the modal form progressive enhancement. Try clicking on the 'contact' or 'survey' links to see the modal form in action

The jQuery plug-in is loading these forms from another page and putting them in the modal. If a user doesn't have javascript enabled, or if there was a problem fetching the remote page, the link will instead operate as normal.
